# AI Usage Verifier
A simple API to handle a large volume of requests with minimal latency.

## Usage
1. Create an instance of the `AIUsageVerifier` class.
2. Call the `handle_request` method to handle a request.
3. Call the `get_latency` method to get the average latency.
4. Call the `get_error_message` method to get an error message.
5. Call the `scale_horizontally` method to scale the API.

## Testing
Run the tests using `pytest` to ensure the API is working correctly.
