# AI Usage Verifier

A Python project to verify AI usage in products.

## Usage

1. Create an instance of the `AIUsageVerifier` class.
2. Add AI usage using the `add_ai_usage` method.
3. Get AI usage using the `get_ai_usage` method.
4. Verify AI usage using the `verify_ai_usage` method.

## Tests

Run the tests using `python -m pytest` in the `tests` directory.
