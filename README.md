# Audit Trail

This project provides a simple audit trail system that allows you to add and query audit records.

## Usage

To use this project, simply create an instance of the `AuditTrail` class and add records to it using the `add_record` method. You can then query the records using the `query_by_model_id` or `query_by_timestamp` methods.
