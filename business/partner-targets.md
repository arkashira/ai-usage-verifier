# Partner Targets

## Integration Roadmap

### 1. **Zapier**
- **Rationale**: Zapier is a leading automation platform that connects thousands of apps. Integrating with Zapier will allow users to automate workflows involving AI usage verification.
- **Free-Tier Limits**: 100 tasks/month.
- **Integration Effort**: S (Zapier provides extensive documentation and APIs).
- **Value-Add**: Solves the job of automating verification processes for non-technical users.

### 2. **Slack**
- **Rationale**: Slack is widely used for team communication. Integrating with Slack will enable real-time notifications and alerts for AI usage verification.
- **Free-Tier Limits**: Limited to 10,000 messages/month.
- **Integration Effort**: M (Slack API is robust but requires careful handling of webhooks and events).
- **Value-Add**: Solves the job of keeping teams informed about AI usage verification status.

### 3. **GitHub**
- **Rationale**: GitHub is the go-to platform for developers. Integrating with GitHub will allow developers to verify AI usage in their repositories.
- **Free-Tier Limits**: Unlimited public repositories, limited private repositories.
- **Integration Effort**: M (GitHub API is well-documented but requires OAuth and webhook setup).
- **Value-Add**: Solves the job of verifying AI usage in code repositories.

### 4. **AWS (Amazon Web Services)**
- **Rationale**: AWS is a leading cloud provider. Integrating with AWS will allow users to verify AI usage in their cloud-based applications.
- **Free-Tier Limits**: Varies by service, generally 12 months free.
- **Integration Effort**: L (AWS API is extensive and requires detailed configuration).
- **Value-Add**: Solves the job of verifying AI usage in cloud environments.

### 5. **Google Cloud Platform (GCP)**
